Add your LiveChat SaaS live chat widget to any WordPress site — no code needed.

== Description ==
Install this plugin, paste your Widget Key (from your LiveChat SaaS dashboard →
Settings → Install), and the chat widget appears on every page of your site.

== Installation ==
1. Upload the `livechat-saas` folder to `/wp-content/plugins/` (or install the ZIP via Plugins → Add New → Upload).
2. Activate the plugin.
3. Go to Settings → LiveChat SaaS, paste your Widget Key + API URL, Save.
4. Done — the widget is live on your site.
