# Product Feature Scope

This project targets a full SaaS live chat platform inspired by LiveChat.com. The features below are part of the product scope and will be delivered module by module.

| Frontend dashboard shell | Pending |

| Advanced LiveChat-style features | Pending |

## Build Phases

1. Frontend shell, auth screens, and dashboard layout.
2. Core chat: conversations, messages, Socket.IO, agent inbox, visitor widget.
3. Productivity: canned responses, tags, file sharing, transfers, ratings, archives.
4. Visitor tracking, CRM, tickets, notifications, transcripts.
5. Widget customization, engagement campaigns, routing, availability, departments.
6. Analytics, reports, exports, goals, ecommerce tracking.
7. Channels and integrations: email, WhatsApp, SMS, Facebook, Instagram, webhooks, SDK/API.
8. AI features, supervision, scheduler, advanced security, SSO, compliance.
