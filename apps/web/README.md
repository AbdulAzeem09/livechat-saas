# Web App

Next.js 15 frontend for the agent dashboard, admin panel, analytics, billing screens, CRM, ticket workspace, and embeddable visitor chat widget.

Planned UI areas:

- `/login` and OAuth callback handling
- `/dashboard` agent inbox, queue, and realtime conversations
- `/contacts` CRM contact management
- `/tickets` ticket system
- `/settings` organization, widget, roles, canned responses, billing
- `/admin` owner/admin controls
- `/analytics` conversation, agent, and revenue reporting
- `/widget` public visitor chat runtime
